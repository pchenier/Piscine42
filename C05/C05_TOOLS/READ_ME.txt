Legende: $$ = chiffre du chapitre ou de l'exercice

Comment utiliser le norm_n_compile
1) Unzip le dossier
2) Placer le dossier dans le fichier intra...... (Au meme endroit que les dossier ex$$
3) Avec le terminal, faire des cd pour se rendre a /intra..../C$$_TOOLS
4) Executer la commande % bash norm_n_compile
- Le code fera la norminette -R ... de chaque fichier et en meme temps
	confirme que les noms de fichier sont les bons
- Le code compilera les main$$.c presents dans le dossier avec son fichier
	respectif avec tous les flags -W avec les noms out$$.
5) Pour tester les fichiers, faite la commande % ./out$$

ps: Le fichier ne verifie pas les fichiers supplementaires presents
	Confirmer avec ls -a
